================================================================================
INFORMAÇÕES DA BASE DE DADOS DE IMAGENS - SENTINEL-2
================================================================================

NOME DA BASE DE DADOS:
  - Sentinel-2 L2A (Level-2A - Processed)
  - Collection ID: sentinel-2-l2a

PROVEDOR:
  - Element 84 (E84)
  - STAC Catalog: https://earth-search.aws.element84.com/v1

SATÉLITE:
  - Sentinel-2A e Sentinel-2B (constelação da Agência Espacial Europeia - ESA)

CARACTERÍSTICAS TÉCNICAS:
  - Resolução espacial: 10m, 20m e 60m (dependendo da banda)
  - Bandas espectrais: 13 bandas (Coastal, Blue, Green, Red, Rededge x3, NIR, NIR2, Watervapor, SWIR x2, SCL)
  - Cobertura global: -56° a +83° latitude
  - Revisita: 5 dias (Sentinel-2A + 2B)
  - Tamanho de cena: ~110km × 110km
  - Formato: GeoTIFF em AWS S3

BANDAS DISPONÍVEIS (geo9.py):
  1. B01 - Coastal Aerosol (60m)
  2. B02 - Blue (10m)
  3. B03 - Green (10m)
  4. B04 - Red (10m)
  5. B05 - Red Edge (20m)
  6. B06 - Red Edge (20m)
  7. B07 - Red Edge (20m)
  8. B08 - Near Infrared (10m)
  9. B8A - Red Edge (20m)
  10. B09 - Water Vapour (60m)
  11. B11 - Short-wave Infrared (20m)
  12. B12 - Short-wave Infrared (20m)

NÍVEL DE PROCESSAMENTO:
  - L2A (Bottom-Of-Atmosphere reflectance)
  - Atmosfericamente corrigido
  - Pronto para análise

COBERTURA DE NUVENS:
  - Filtro aplicado: máximo 50% de cobertura de nuvens (configurável)

DADOS HISTÓRICOS:
  - Disponível desde: Dezembro de 2015
  - Cobertura contínua global desde 2016

APLICAÇÕES:
  - Monitoramento agrícola
  - Análise de uso e cobertura da terra
  - Detecção de mudanças
  - Mapeamento de água
  - Super-resolução de imagens
  - Análise ambiental

ACESSO:
  - Via STAC API (SpatioTemporal Asset Catalog)
  - Dados armazenados em AWS S3
  - Acesso público e gratuito
  - Download via biblioteca 'cubo' (em geo9.py)

DOCUMENTAÇÃO:
  - ESA: https://sentinel.esa.int/web/sentinel/missions/sentinel-2
  - AWS: https://registry.opendata.aws/sentinel-2-l2a/
  - Element 84 STAC: https://earth-search.aws.element84.com/v1
